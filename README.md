# webpackDemo

> This project is a simple HTML list of To Do tasks. Built using webpack and served by a webpack server.
> For this part of this project we add this functionalities to the list:
- add task
- remove task
- update task


## Built With

- Html, Js, Css.
- Webpack.
- Linters.
